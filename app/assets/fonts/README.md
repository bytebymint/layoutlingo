# Bundled RTL Fonts

These fonts are embedded into translated PDFs so complex-script output does
not depend on fonts installed on the application host.

- `NotoSansArabic.ttf`: Arabic and Farsi
- `NotoSansHebrew.ttf`: Hebrew
- `NotoNastaliqUrdu.ttf`: Urdu
- `NotoSans.ttf`: Latin, numbers, punctuation, and symbol fallback
- `NotoSansSymbols.ttf`: arrows and technical symbols fallback
- `NotoSansMath.ttf`: mathematical notation fallback

The files are distributed by the official Google Fonts repository and are
